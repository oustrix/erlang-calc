# Erlang Calc

Erlang Calc — десктопное приложение на базе Wails (Go + Svelte), которое рассчитывает основные показатели Erlang B и Erlang C для call-центров и других систем с очередями. Все вычисления выполняются на стороне Go, а Svelte отвечает за отображение формы ввода и результатов.

## Возможности

- Расчёт вероятности блокировки (Erlang B) и ожидания (Erlang C) по заданной нагрузке и числу операторов.
- Дополнительные метрики: фактически принятая нагрузка, текущая и эффективная утилизация операторов.

## Предпосылки

- Go 1.23+
- Node.js 18+ и npm
- Wails CLI `v2.11.0` (устанавливается через `go install github.com/wailsapp/wails/v2/cmd/wails@latest`)

## Установка зависимостей

```bash
# Backend
go mod tidy

# Frontend
cd frontend
npm install
```

## Режим разработки

```bash
wails dev
```

Команда запускает дев-сервер Vite с горячей перезагрузкой (порт определяется автоматически) и Go-бэкенд. Во время разработки интерфейс можно открыть в браузере или встраиваемом окне Wails.

## Сборка продакшн-версии

```bash
wails build
```

Результат будет сохранён в директорию `build/` для соответствующей платформы.

## Использование

1. Введите offered load в эрлангах (например, произведение интенсивности поступления заявок на среднее время обслуживания).
2. Укажите количество доступных операторов.
3. Нажмите «Рассчитать», чтобы получить блокировку/ожидание и вспомогательные метрики.
